# Movie-Tracker

Movie-Tracker é um sistema para gerenciamento de filmes para assistir.

## Tecnologias utilizadas

Frontend:
- Angular

Backend:
- Java Spring Boot

Banco de dados:
- MySQL

## Funcionalidades

1. Adicionar filmes
2. Listar filmes
3. Marcar filmes como assistidos
4. Avaliar filmes

## Estrutura do projeto

frontend/
Aplicação Angular responsável pela interface do usuário.

backend/
API REST desenvolvida com Spring Boot.

database/
Scripts SQL para criação das tabelas no MySQL.
