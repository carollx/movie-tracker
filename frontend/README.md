# Frontend do Projeto Movie-Tracker (Angular)

Frontend da aplicação Movie Tracker desenvolvido com Angular, para consumir a API do backend Spring Boot.

## Pré-requisitos

- Node.js 18+
- Angular CLI 16+

## Instalação

1. Instale as dependências:

npm install

## Executar a Aplicação

ng serve

A aplicação estará disponível em: `http://localhost:4200`

## Funcionalidades

- Listar filmes
- Adicionar novo filme
- Marcar filme como assistido
- Avaliar filmes

## Tecnologias Utilizadas

- **Angular 16**
- **TypeScript**
- **RxJS**
- **Angular CLI**

## Conexão com o Backend

Backend deve estar rodando em `http://localhost:8080` para que a API funcione corretamente.
